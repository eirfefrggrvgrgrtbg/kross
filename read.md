# CineOnline - Онлайн-кинотеатр

Мобильное приложение для любителей кино, созданное с использованием Flutter.

## Основные сведения

**CineOnline** — это приложение, которое позволяет пользователям просматривать каталог фильмов, изучать подробную информацию о них и управлять собственными коллекциями ("Избранное" и "Посмотреть позже").

## Функциональность

*   **Аутентификация**: Полноценная система регистрации и входа.
*   **Главный экран**: Лента фильмов с возможностью перехода к деталям.
*   **Детали фильма**: Просмотр описания, рейтинга, жанров и длительности.
*   **Персонализация**:
    *   Добавление фильмов в список "Избранное".
    *   Планирование просмотра (список "Посмотреть позже").
*   **Профиль пользователя**: Настройки аккаунта и приложения.
*   **Темы оформления**: Поддержка светлой и темной тем.
*   **Локальная БД**: Все данные о фильмах и предпочтениях пользователя хранятся локально с использованием Drift (SQLite).

## Технологический стек

*   **Язык**: Dart
*   **Фреймворк**: Flutter
*   **Управление состоянием**: BLoC / Cubit (`flutter_bloc`)
*   **База данных**: Drift (ранее Moor)
*   **Внедрение зависимостей**: Injectable & GetIt
*   **Библиотеки**:
    *   `equatable` для сравнения объектов.
    *   `path_provider` для работы с файловой системой.
    *   `cached_network_image` для эффективной загрузки изображений.

## Запуск проекта

### Требования
*   Установленный Flutter SDK
*   Эмулятор Android/iOS или физическое устройство

### Инструкция

1.  **Получение зависимостей**:
    ```bash
    flutter pub get
    ```

2.  **Генерация кода**:
    Так как проект использует `drift` и `injectable`, необходимо сгенерировать файлы перед запуском:
    ```bash
    dart run build_runner build --delete-conflicting-outputs
    ```

3.  **Запуск приложения**:
    ```bash
    flutter run
    ```

## Структура папок

*   `lib/bloc/` — BLoC-компоненты для сложной логики.
*   `lib/cubit/` — Cubit-компоненты для простого управления состоянием.
*   `lib/data/` — Слой данных: БД, репозитории, моки.
*   `lib/di/` — Конфигурация Dependency Injection.
*   `lib/models/` — Описание моделей данных.
*   `lib/pages/` — Все экраны приложения.
*   `lib/widgets/` — Общие UI-компоненты.
